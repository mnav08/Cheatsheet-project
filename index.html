Display & Positioning Properties Reference

Display Property

The display property specifies the display behavior (the type of rendering box) of an element.

Value Description Syntax
Inline Displays an element as an inline element (like span). Any height and width properties will have no effect. This is default. display: inline;
Block Displays an element as a block element (like p). It starts on a new line, and takes up the whole width display: block;
Inline-Block Displays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width values display: inline-block;

Position Property

The position property specifies the type of positioning method used for an element (static, relative, absolute, fixed, or sticky).

Value Description Syntax
static Default value. Elements render in order, as they appear in the document flow position:static;
absolute The element is positioned relative to its first positioned (not static) ancestor element position: absolute;
